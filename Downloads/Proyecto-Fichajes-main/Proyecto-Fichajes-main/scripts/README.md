# Scripts de utilidad

Scripts de setup, debugging y testing. No son necesarios para el funcionamiento normal del sistema.

- **crear_usuarios_bcrypt.py**: Crea usuarios de prueba directamente con bcrypt
- **test_telegram.py**: Verifica la configuración del token de Telegram
- **verificar_usuarios.py**: Lista usuarios en MongoDB
- **resetear_admin.py**: Resetea el administrador a valores por defecto
- **crear_usuariosPrueba.py**: Crea usuarios de prueba con estructura completa

Para ejecutar cualquiera:
```powershell
python .\scripts\nombre_del_script.py
```
